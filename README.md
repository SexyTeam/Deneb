# Deneb

A Crystal PVP Utility mod for Minecraft 1.12.2 anarchy server.

## How to Build

1.Open cmd or powershell.

2.Use command cd <project path>

3.Use command gradlew build (Once fialed ,just try again)

4.Move jar to .minecraft/mods/
